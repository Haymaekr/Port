Brainwave Controlled Wheelchair

Previous Next

The objective of this project is to analyse the EEG signals from the Motor Cortex of the brain for different motor imagery, denoise the signal, extract appropriate features and ultimately control the wheelchair based on those features. However, we had to limit the use to just two hardware units- one, to pick up the EEG signals and wirelessly transmit it via Bluetooth to the processing element which forms the second hardware unit.

The project concludes into a functional Brain Wave Controlled Wheelchair. Right, Left commands given by a disabled person is deciphered by embedded algorithms and suitably executed with an accuracy of about 82%. By selecting suitable features and training data for design, cost effectiveness of overall project is optimized. The designed ‘Brain Wave Controlled Wheelchair’ provides control of motion and direction of wheelchair by mind’s attention and meditation level and intent for movement.

Password Checker

Previous Next

A Python program that verifies the strength of your password. The password is converted into its SHA-1 hash value and split into two parts. The first four digits are tallied with the database from the pawned password and then internally matched to see if the entered password was hacked or obtained from a databreach.

Since entering the password directy into the website can be traced and leades to potential databreaches, the password is handled within the code. Therefore, no external source can access it.
